# DummyJson Angular

## About the project 

This project provides frontend for [DummyJson Api](https://dummyjson.com/docs/auth). I have covered both authenticated and unauthenticated endpoints for learning.

## Topics Covered

1. Angular Standalone
2. Angular Guards 
3. Interceptors
4. JWT in Cookies
